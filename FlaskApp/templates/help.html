How to Use Control Panel

Welcome to Haz Bot! This guide will help you navigate the features and functionalities of the website with ease.

1. Navigation Menu

Use the navigation bar at the top to access different sections like Home, Help, and About.

2. Homepage Overview

The homepage contains feature cards for easy interaction. Click on any card to explore its features.

  • Rover Settings: Select mission
  • Rover Status: states if the rover is Disarmed or Armed
  • Rover Messages: shows received data log informaiton back from the rover for analysis and diagnostics

3. Button Explanations

  • Arm: Arms the rover, enabling it for operation. Click this to prepare the rover for use.
  • Disarm: Disarms the rover, ensuring it is safely deactivated. Use this to stop operations.
  • Start Mission: Initiates a pre-programmed mission for the rover. Ensure the rover is armed before starting.
  • Reset Mission : It immediately resets any active mission and clears it from the rover, allowing a new mission to be uploaded
  • Auto Mode: Activates autonomous mode, allowing the rover to navigate without manual control and for missions to be uploaded if needed.
  • Manual Mode: Deactivates autonomous mode, switching control back to manual operation.

4. RC Controller

An RC controller should be attached to the rover. Once manual mode is selected and the rover is armed, you can use the controller to drive the rover.

5. Troubleshooting

If issues arise, refresh the page or check your internet connection. Contact support if needed.
